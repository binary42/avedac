Automated Visual Event Detection

Event Table
The Event table is a spreadsheet display of AVED events where each row defines a single AVED event.

Columns can be sorted in increasing or decreasing order by clicking any header label







Each table row defines a single AVED event



Each column displays one type of AVED data as defined as follows:

Object ID Max Size Start Timecode End
Timecode		 Start
Frame		 End
Frame		 Thumbnail
Event Object Identifier Largest square pixel size of the event found over its duration  Starting time code of the event, or missing if no timecode used Ending time code of the event, or missing if no timecode used Starting frame number of the event Ending frame number of the event Event image with the largest size found over its duration. This is generally the best representation of an event.


Event table selection


Events can be selected in the table according to the following:

Operation Mouse Action Keyboard Action
Select single row. Click. Up Arrow or Down Arrow.
Extend contiguous selection. Shift-Click or Drag over rows. Shift-Up Arrow or Shift-Down Arrow.
Add row to selection/toggle row selection. Mac:Command-Click 
Other: Control-Click		 Other only: Move lead selection with Control-Up Arrow or Control-Down Arrow, then use Space Bar to add to selection or Control-Space Bar to toggle row selection.

Once events are selected, delete events with Control-Click, then Delete, or Delete all
