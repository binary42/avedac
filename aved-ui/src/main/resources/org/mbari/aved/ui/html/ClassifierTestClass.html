AVEDac Classifier Step 3. Test Class

Testing a class

Testing a class is a method that test a 10% random sample of the class against a training library. If less than 10 images exist in a class, all the images are tested.  This is an optional step, but useful to determine how effective a class in discriminating itself.  To test a class, select the class to test.  This class to test must exist in the training library.  Select the training library, then click run.  


Once the testing is complete, you will see the results from the training in the form of a Confusion Matrix:

This will give you an indication of how well your class is performing. This table can be exported to Excel by right clicking on it, then selecting Export to Excel.
